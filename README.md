# Spotlight Game Rules

This repository hosts the game rules page for the Spotlight Bot on [Blaze.stream](https://blaze.stream).

## About

The Spotlight Bot is a game created by jt_squared2 to bring more attention and support to channels on Blaze streaming platform.

## View the Rules

Visit the live page at: https://jtSquared.github.io/spotlight-game-rules/

## Game Features

- Compete for the spotlight by streaming
- Earn points through followers, subscribers, and donations
- Win prizes including votes and backstage passes
- Weekly competitions with rankings

Created by jt_squared2 for the Blaze.stream community.
